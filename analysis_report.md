# Sales Data Analysis Report

## 1. Objective
The objective of this task is to analyze a sales dataset using Python and Pandas, handle missing values, and calculate basic statistical metrics.

## 2. Tools Used
- Python
- Pandas library

## 3. Dataset Description
The dataset contains sales-related information with numerical and categorical columns. Some columns contain missing values.

## 4. Data Loading
The dataset was loaded using the `read_csv()` function from the Pandas library. The first five rows were displayed to understand the structure.

## 5. Data Inspection
- Number of rows and columns were identified.
- Data types of each column were checked.
- Missing values were detected using `isnull()`.

## 6. Handling Missing Values
Missing numerical values were filled using the mean of their respective columns to maintain data consistency.

## 7. Metrics Calculated
The following statistical metrics were calculated for numerical columns:
- Average (Mean)
- Maximum value
- Minimum value

## 8. Conclusion
The sales dataset was successfully analyzed using Pandas. Missing values were handled appropriately, and meaningful statistical insights were extracted.
